WINDOWS-SPECIFIC TIPS:
═══════════════════════════════════════════════════════════════════

PowerShell Execution Policy:
  If .ps1 scripts won't run:
  Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Virtual Environment Activation:
  PowerShell: .venv\Scripts\Activate.ps1
  Command Prompt: .venv\Scripts\activate.bat
  NOT: source .venv/bin/activate (that's for Mac/Linux)

File Paths:
  • Use backslash \ in Windows commands
  • Use forward slash / in PostgreSQL commands
  • Use forward slash / in .env file

Port Conflicts:
  Find process: netstat -ano | findstr :8545
  Kill process: taskkill /PID <PID> /F

Firewall:
  If services can't connect, add inbound rules for:
  • Port 3000 (Frontend)
  • Port 8000 (Backend)
  • Port 8545 (Ganache)
